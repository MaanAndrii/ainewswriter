# AI Newswriter

Інструмент для підготовки публікацій українського онлайн-медіа на базі AI. Приймає вхідний матеріал будь-якою мовою і повертає готові заголовки, ліди, статтю та Facebook-допис у форматі JSON.

## Можливості

- **Рерайт і підготовка новин** — вхідний матеріал будь-якою мовою перетворюється на готову публікацію українською
- **Кілька форматів виводу** — заголовки, ліди, стаття, Facebook-допис
- **Гнучкі параметри обробки:**
  - Глибина рерайту (0–3): мінімальні правки → повна переробка структури
  - Тональність: нейтральна, інтригуюча, емоційна, SEO
  - Додаткові інструкції до кожного запиту
  - Веб-пошук для актуалізації фактів (Anthropic і Gemini)
  - Органічне вписування джерела в перший абзац
- **Підтримка чотирьох AI-провайдерів** — Anthropic, xAI, Google Gemini, Mistral
- **Адмін-панель** — управління моделями, ключами, промтами, статистика, імпорт/експорт
- **Детальні логи** — вартість, токени, кеш-статус, час відповіді по кожному запиту
- **Prompt caching** — кешування системного промту для Anthropic (знижує вартість)

## Конфігурація

Всі секрети зберігаються у файлі `.env.local` у корені проєкту.

**Формат файлу:**
```bash
# Коментар
KEY=value
```

**Пріоритет завантаження:** змінні середовища сервера → `.env.local` → значення за замовчуванням.

Щоб вказати інший шлях до env-файлу, встановіть змінну середовища `APP_ENV_FILE`.

### Доступні параметри

| Змінна | Опис | За замовчуванням |
|--------|------|-----------------|
| `ANTHROPIC_API_KEY` | Ключ Anthropic API | — |
| `XAI_API_KEY` | Ключ xAI API | — |
| `GEMINI_API_KEY` | Ключ Google Gemini API | — |
| `MISTRAL_API_KEY` | Ключ Mistral API | — |
| `ADMIN_PASSWORD` | Пароль адмін-панелі | `change-me-now` |
| `LOG_PASSWORD` | Пароль log viewer | значення `ADMIN_PASSWORD` |
| `CORS_ALLOWED_ORIGINS` | Дозволені CORS-домени (через кому) | `*` |

### Приклад .env.local

```bash
ADMIN_PASSWORD=my-secure-password
LOG_PASSWORD=another-password

ANTHROPIC_API_KEY=sk-ant-api03-...
XAI_API_KEY=xai-...
GEMINI_API_KEY=AIzaSy...
MISTRAL_API_KEY=...

CORS_ALLOWED_ORIGINS=http://localhost:3000,https://my-newsroom.com
```

## Підтримувані AI-провайдери

| Провайдер | `provider` | API-ключ | Формат API |
|-----------|-----------|----------|------------|
| Anthropic (Claude) | `anthropic` | `ANTHROPIC_API_KEY` | Messages API v1 |
| xAI (Grok) | `xai` | `XAI_API_KEY` | OpenAI-compatible |
| Google Gemini | `gemini` | `GEMINI_API_KEY` | Generative Language API |
| Mistral | `mistral` | `MISTRAL_API_KEY` | OpenAI-compatible |

### Додавання моделей

Моделі налаштовуються в адмін-панелі (вкладка **Налаштування AI**) або напряму в `prompts.json` → `default_settings.models`:

```json
{
  "id": "claude-sonnet-4-6",
  "label": "Sonnet 4.6 — баланс / рекомендовано",
  "provider": "anthropic",
  "inp": 3.0,
  "out": 15.0,
  "web_search": true
}
```

| Поле | Опис |
|------|------|
| `id` | Ідентифікатор моделі на боці провайдера |
| `label` | Назва у випадаючому списку UI |
| `provider` | `anthropic` / `xai` / `gemini` / `mistral` |
| `inp` | Вартість вхідних токенів, $ за 1M |
| `out` | Вартість вихідних токенів, $ за 1M |
| `web_search` | `true` — увімкнути опцію веб-пошуку для цієї моделі |

## Веб-пошук

| Провайдер | Підтримка | Реалізація |
|-----------|-----------|------------|
| Anthropic | ✅ | `web_search_20250305` tool |
| Google Gemini | ✅ | Google Search Grounding (`google_search` tool) |
| xAI | ❌ | `search_parameters` deprecated, Agent Tools API потребує multi-turn |
| Mistral | ❌ | Потребує окремого Agents API або зовнішнього пошуку |

Веб-пошук активується чекбоксом **«Веб-пошук»** в інтерфейсі. Якщо модель не підтримує пошук — чекбокс автоматично блокується.

Чи дійсно модель використала пошук видно у рядку вартості після запиту (мітка **+ веб-пошук 🔍**), а не лише стан чекбоксу.

## Адмін-панель

**URL:** `/admin` або `/admin/admin.php`
**Авторизація:** пароль з `ADMIN_PASSWORD`

### Вкладки

#### Налаштування AI
- Перегляд та оновлення API-ключів (маскуються у відображенні, зберігаються в `.env.local`)
- Додавання, редагування та видалення AI-моделей
- Зміна порядку моделей перетягуванням

#### Промти і параметри
- Редагування системного промту
- Налаштування лімітів генерації: кількість заголовків, лідів, максимальна довжина статті та Facebook-допису, мін/макс довжина ліду
- Редагування полів профілю промту: тональності, глибини рерайту, стилі Facebook, правила вписування джерела, інструкції веб-пошуку
- Кнопка **«Відновити стандартні»** — скидає до значень з `prompts.json`

#### Статистика
- Загальна кількість запитів, витрати, токени
- Розбивка по моделях
- Денна статистика (останні 14 днів)

#### Безпека
- Зміна пароля адмін-панелі
- Зміна пароля log viewer

#### Логи
- Останні 100 запитів
- Посилання на повний Log Viewer

#### Імпорт / Експорт
- **Експорт** — зберегти поточні налаштування в JSON-файл (без API-ключів)
- **Імпорт** — відновити налаштування з резервної копії (вибірково: моделі, промти, системний промт, ключі)

## API

### POST /api/proxy

Проксі-запит до AI-провайдера.

**Тіло запиту:**
```json
{
  "prompt": "текст для обробки",
  "model": "claude-sonnet-4-6",
  "webSearch": 1,
  "systemPromptOverride": ""
}
```

| Поле | Тип | Опис |
|------|-----|------|
| `prompt` | string | Вхідний промт, максимум 30 000 символів |
| `model` | string | ID моделі з налаштувань |
| `webSearch` | int | `1` — увімкнути веб-пошук, `0` — вимкнути |
| `systemPromptOverride` | string | Замінює системний промт (необов'язково) |

**Відповідь (200 OK):**
```json
{
  "ok": true,
  "text": "{ \"headlines\": [...], \"article\": \"...\" }",
  "usage": {
    "input_tokens": 1234,
    "output_tokens": 567,
    "cache_creation_input_tokens": 100,
    "cache_read_input_tokens": 0
  },
  "meta": {
    "provider": "anthropic",
    "model": "claude-sonnet-4-6",
    "web_search_used": false
  }
}
```

**Відповідь (помилка):**
```json
{
  "error": "опис помилки",
  "type": "тип_помилки",
  "code": 429,
  "provider": "anthropic",
  "model": "claude-sonnet-4-6"
}
```

### GET /api/settings

Повертає конфігурацію для UI: список моделей, системний промт, профілі промтів, поточна модель за замовчуванням.

## Файли сховища

Створюються автоматично в папці `storage/`. Переконайтесь що вона доступна для запису `www-data`.

### storage/requests.log

Лог у форматі JSONL (один запис на рядок). Ротується при досягненні 5 МБ → `requests.log.1`.

```jsonc
{
  "v": 2,
  "date": "2025-05-15",
  "time": "14:32:45",
  "model": "claude-sonnet-4-6",
  "inp": 1234,           // вхідні токени
  "out": 567,            // вихідні токени
  "cache_write": 100,    // записано в кеш
  "cache_read": 0,       // прочитано з кешу
  "cost": 0.0057,        // вартість у USD
  "duration": "2.45",    // час відповіді в секундах
  "prompt_len": 8900,    // довжина промту
  "web": false,          // чи використано веб-пошук
  "cache_status": "cache-write"  // cache-hit | cache-write | no-cache
}
```

### storage/api_responses.json

Останні 5 повних відповідей від API (для діагностики). Зберігаються як успішні так і помилкові відповіді, перші 8 000 символів тіла.

## Налаштування промтів

Файл `prompts.json` задає стандартні значення для системного та користувацького промту.

### Системний промт (`system_prompts.default`)

Інструкція для моделі: роль редактора українського медіа, мовні вимоги (Правопис 2019), редакційні стандарти (перевернута піраміда, нейтральність), формат відповіді (тільки JSON).

### Профіль юзер-промту (`user_prompt_profiles.default`)

Шаблон промту з плейсхолдерами, які замінюються перед відправкою:

| Плейсхолдер | Значення |
|-------------|----------|
| `{{headlines_count}}` | Кількість заголовків |
| `{{leads_count}}` | Кількість лідів |
| `{{lead_min_chars}}` / `{{lead_max_chars}}` | Межі довжини ліду |
| `{{article_max_chars}}` | Максимум символів статті |
| `{{facebook_max_chars}}` | Максимум символів Facebook-допису |
| `{{tone_short}}` | Коротка інструкція тональності |
| `{{depth_short}}` | Коротка інструкція глибини рерайту |
| `{{source_ref}}` | Назва джерела для органічного вписування |
| `{{fb_style_rule}}` | Правило стилю Facebook-допису |

Усі ліміти та тексти редагуються в адмін-панелі (вкладка **Промти і параметри**) без зміни файлу.

## Розгортання на Raspberry Pi

Детальна покрокова інструкція — у файлі [RASPBERRY_PI_SETUP.md](RASPBERRY_PI_SETUP.md).

**Короткий огляд:**

1. Встановіть пакети:
   ```bash
   sudo apt install -y nginx php-fpm php-curl git
   ```

2. Клонуйте репозиторій у `/var/www/ainewswriter`

3. Налаштуйте nginx (конфіг з таймаутами для reasoning-моделей):
   ```nginx
   location ~ \.php$ {
       include snippets/fastcgi-php.conf;
       fastcgi_pass unix:/run/php/php8.4-fpm.sock;
       fastcgi_read_timeout 300;
       fastcgi_send_timeout 300;
   }
   ```

4. Збільшіть таймаут PHP-FPM (`/etc/php/8.4/fpm/pool.d/www.conf`):
   ```ini
   request_terminate_timeout = 300
   ```

5. Створіть `.env.local` з API-ключами та паролем адміна

> **Чому 300 секунд?** Reasoning-моделі (grok-reasoning, Claude з extended thinking) можуть думати 60–180 сек. Без збільшення таймауту nginx поверне `504 Gateway Timeout` ще до того як модель відповість.

## Безпека

- API-ключі зберігаються в `.env.local` (gitignored, права `0600`)
- Файл налаштувань `settings_store.php` не доступний через веб (заблокований роутером)
- Сесійна авторизація для адмін-панелі та log viewer (окремі паролі)
- CORS-обмеження на рівні проксі (налаштовуються через `CORS_ALLOWED_ORIGINS`)
- Вхідний промт обмежений 30 000 символів
- Паролі зберігаються в `.env.local` у відкритому вигляді — використовуйте сильні паролі і закривайте адмін-панель від зовнішнього доступу (наприклад, через nginx IP-фільтрацію)

## Залежності

**PHP (сервер):**
- PHP 8.0+ (тестувалось на 8.4)
- Розширення: `php-curl`
- Розширення `php-mbstring` — опціонально (є fallback на `strlen`/`substr`)

**Веб-сервер:**
- nginx + PHP-FPM (рекомендовано)
- або `php -S localhost:PORT` (розробка)

**Браузер:**
- Будь-який сучасний браузер (ES6+)

## Діагностика

### Перевірка nginx і PHP-FPM

```bash
sudo nginx -t
sudo tail -n 100 /var/log/nginx/error.log
sudo journalctl -u php8.4-fpm -n 100 --no-pager
```

### Перевірка синтаксису PHP

```bash
cd /var/www/ainewswriter
find . -maxdepth 3 -name "*.php" | xargs -n1 php -l
```

### Типові помилки

| Помилка | Причина | Вирішення |
|---------|---------|-----------|
| `HTTP 500` без тексту | `json_encode()` не вдалось (некоректний UTF-8 у відповіді API) | Оновіться до останньої версії — виправлено |
| `HTTP 504` | Gateway timeout — reasoning-модель думає довше ніж таймаут nginx | Збільшіть `fastcgi_read_timeout` до 300 секунд |
| `Call to undefined function mb_substr()` | Відсутнє розширення `php-mbstring` | Оновіться — є fallback на `substr`; або `apt install php-mbstring` |
| `Помилка розбору відповіді` | Mistral/xAI повертають реальні `\n` всередині JSON-рядків | Оновіться — виправлено |
| `Немає доступних моделей` | `prompts.json` пошкоджений або порожній `settings_store.php` | Перевірте JSON через `php -r "echo json_last_error_msg();"` |
| Адмін-панель не зберігає | `settings_store.php` або папка не доступна для запису | `sudo chown www-data:www-data /var/www/ainewswriter` |
